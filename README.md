# VroomUI 🚗💨

VroomUI is a modern, high-performance web application built with **Angular 21**, **Tailwind CSS**. It features a sleek, dark-themed interface designed for students, instructors, and administrators to manage educational scenarios and assignments.

## 🚀 Key Features

- **Auth & Security**: Robust authentication guards and secure login flow.
- **Student Dashboard**: Personalized dashboard for students to track progress and assignments.
- **Instructor Portal**: Comprehensive tools for instructors to manage students, scenarios, and view analytics.
- **Admin Management**: Administrative controls for system-wide configuration.
- **Modern UI**: Responsive design with a glassmorphism aesthetic, custom animations, and a sidebar navigation.
- **Dark Mode First**: Optimized for a high-contrast dark theme.

## 🛠️ Tech Stack

- **Framework**: [Angular 21](https://angular.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide Angular](https://lucide.dev/)
- **Build Tool**: [Angular CLI](https://github.com/angular/angular-cli)
- **Testing**: [Vitest](https://vitest.dev/)

## 📦 Installation & Setup

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Steps
1. **Clone the repository**:
   ```bash
   git clone https://github.com/theshamkhi/VroomUI.git
   cd VroomUI
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```
   Navigate to `http://localhost:4200/` in your browser.

## 🛠 Development Commands

### Code Scaffolding
Generate a new component:
```bash
ng generate component component-name
```

### Building
Compile the project for production:
```bash
ng build
```
The artifacts will be stored in the `dist/` directory.

### Testing
Run unit tests with Vitest:
```bash
ng test
```
